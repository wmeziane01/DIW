Cumulative Layout Shift (CLS)

El Cumulative Layout Shift (CLS) es una métrica de rendimiento web que mide la estabilidad visual de una página. Si el contenido de una página se mueve inesperadamente mientras el usuario interactúa con ella, genera una mala experiencia de usuario.

Por ejemplo, si un botón cambia de posición debido a que una imagen o un anuncio tarda en cargar, el usuario podría hacer clic en un lugar no deseado.

En el ejemplo anterior, el botón se mueve cuando la imagen tarda en cargarse, lo que causa un layout shift. Para evitar esto, debemos reservar espacio para las imágenes u otros elementos de carga lenta usando propiedades como width, height o, si conocemos la relación de aspecto de la imagen, aspect-ratio.

Para comprobar este efecto, puedes usar las DevTools del navegador y simular una conexión lenta en la página modificando la opción throttling en la pestaña Network.
